# Youtube search app

Youtube search app made with react.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Node
Git
Your favorite editor!

### Installing

In order to install you should run the following commands:

git clone https://github.com/JosueCuberoSanchez/youtube_seach_app.git
cd [generated-directory]
npm install

## Built With

* [Boostrap](https://getbootstrap.com/) - The styles library used.
* [Gulp](https://gulpjs.com/) - Build, server simulation and deploy tool used.
